

Check NECR (Noise Equivalent Count Rate) for Philips VEREOS PET system
----------------------------------------------------------------------

See:
Salvadori J, Labour J, Odille F, Marie PY, Badel JN, Imbert L, Sarrut D.
Monte Carlo simulation of digital photon counting PET.
EJNMMI Phys. 2020 Apr 25;7(1):23.
doi: 10.1186/s40658-020-00288-w.


Run the simulation with:

Gate mac/mac_NECR.mac


The analysis can be performed with the following python script (that need GateTools):
./runAnalysis.py output
